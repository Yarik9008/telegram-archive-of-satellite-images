Доработки ПО:
    Верхний уровень:
        - расчет азимута ориентира по двум точкам через веб интерфейс 
        - выбор набора спутников для рассчета расписания 
        - оптимизировать пересекающиеся пролеты
        - вывод в телеметрию оставшегося места на диске
        - возможность через веб интерфейс регулировать gain програмного усиления 
        - прредпросмотр принятых снимков через веб интерфейс и возможность скачать zip архив со всеми данными 
        - чистка старых пролетов при полностью забитом диске 
        - отправка принятых снимков в автоматическом режиме в телеграмм канал-архив 
        - создание карты энергетической обстановки по логам принятых снимков 
        - починить логи которые выводяться в веб интерфейс
        - реализовать вывод заряда батареи в веб интерфейс (вольтаж, ток, процент заряда АКБ)
        - реализовать мониторинг температуры процессора встроенными сенсерами для контроля температуры внутри корпуса (особенно актуально будет летом), так же логи температуры должны сохраняться в лог файл.
        - реализовать монитроринг температуры аккумулятора
    
    Деродирование:
        - автоматический разворот снимков в нормальную ориентацию 
        - автоматическое создание продукта L1B для спутников NOAA 
        - автоматическое создание продукта grotiff

    Нижний уровень:
        - переписать общение нижнего уровня с верхнем
        - реализовать режимы работы нормальный, экономный, супер-экономный
        - реализовать поддержку датчиков температуры для мониторинга температуры платы с преобразователями и мониторинга температуры аккумулятора

    1) кожух патч антенны (оставляем без кожуха / я рисую модельку и передаю ее кому то)
    2) кожух блока электроники тот что есть сейчас имеет пробоину и по месту заклеен скотчем, из за этого щели 
    3) корпус (из листового металла - 30 т.р. / фрезерованый - 100-120 т.р. точную цену надо запросить на заводе через производственный отдел)
    4) упаковка (либо отдать Зайцману либо ВИ тот образец который стоит сейчас на крыше для создания упаковки)
    5) есть желание обновить платы нижнего уровня под более оптимизированные, но это стоит порядка 20-30 т.р., сейас есть в запасе платы без компонентов для трек комплксов.
